# MO-IT101---Computer-Programming-1

## MotorPH Payroll System

The MotorPH Payroll System is a console-based application developed for MO-IT101 – Computer Programming 1. The system automates employee payroll computation using text-based input and output.

Employee salary rates are stored on a monthly basis, but the system computes the employee’s weekly salary based on the total number of hours worked within a week. Employee attendance is calculated using daily log-in and log-out records while considering official office hours and a 10-minute grace period for tardiness.

Government-mandated deductions such as SSS, PhilHealth, Pag-IBIG, and withholding tax are computed on a monthly basis. These deductions are then converted proportionally to a weekly amount to determine the employee’s net weekly salary.

All payroll details, including employee information, hours worked, gross weekly salary, monthly deductions, and net weekly salary, are displayed in a clear and organized format in the console.
